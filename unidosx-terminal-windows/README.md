# KitsuVM // UNiDOsX's Terminal Windows

Windows backends for the terminal module of the UNiDOsX Engine.

## Backends

- **Win32 Backend**: Uses the Win32 Console API for input and output operations.
- **VT Backend**: Uses the Win32 Console API only for input operations, while leveraging VT sequences for output operations.
